title: Public exports
description: Categorized list of the public exports from @peteqian/browser-agent-sdk.
---

Everything below is exported from the package root `@peteqian/browser-agent-sdk`.

<Callout type="warn">
Anything imported from the `@peteqian/browser-agent-sdk/internal` subpath (raw CDP,
profile management, DOM serializer, action executor, prompt internals) carries **no
stability guarantee** and may change without a major version bump.
</Callout>

## Core

| Export | Kind | Description |
| --- | --- | --- |
| `Browser` | class | High-level browser handle; owns launch/connect lifecycle. |
| `BrowserOptions` | type | Constructor options for `Browser` (incl. `fingerprint`, `humanize`, `proxyPool`, `cdpUrl`). |
| `BrowserSession`, `Page` | class | Lower-level session and page handles. |
| `Agent` | class | Task runner wrapping the decision loop. |
| `runTask` | function | One-shot entry point. See [runTask options](/docs/reference/run-task). |
| `SimpleAgentOptions`, `AgentProviderOptions` | type | Option shapes for `Agent` / `runTask`. |
| `AgentController` | class | The underlying agent loop controller. |
